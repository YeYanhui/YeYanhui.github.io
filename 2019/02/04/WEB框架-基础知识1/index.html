app目录里面的文件含义：

  • migrations 文件里面的文件:差异化，每次记录修改表结构的记录。 修改数据表结构
  • init.py文件的作用，在Python3中可有可无，但是在Python2中必须要有。
  • admin ： Django 为我们提供的后台管理。
  • models： ORM，写指定的类，通过命令可以创建数据库结构。
  • test.py: 单元测试。
  • views : 业务代码。
